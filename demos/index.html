µCSS™ Demos

Interactive samples built with µCSS™ 2, µPS and µAU — classic desktop-demo effects (Oxyd lineage).

  • Glittery — mouse-move sparkles with random radial offset (sprite strip + GlitterySprite hook)
  • FlyEx — Atari ST fly swatter remake (FlyEx macros + sound atlas)
  • Buttons — µCSS™ 1 buttons.psd: two PSD layouts → sprite atlas → CSS (ButtonAndIconCreator)

Source: github.com/mamekudz/microCSS · npm: gulp-mu-css
