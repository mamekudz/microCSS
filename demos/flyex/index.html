FlyEx — fly swatter

A modern remake of a classic Atari ST desktop gimmick: a fly crosses the screen, you swat it with the cursor. The animation uses the original hand-pixelated sprites from that era — here packed into DSD strips and driven by µCSS (FlyEx generates 80+ CSS rules from one atlas; µAU bundles buzz and swat sounds via the skin manifest).

Tap/click to start audio · Klick auf die Fliege zum Klatsch · Atari-PD-Gimmick-Nachbau · AiDPix / Oxyd · µCSS 2 + µAU
